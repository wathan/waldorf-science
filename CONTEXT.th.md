# Thai Glossary — Waldorf Science Curriculum Reform

Canonical EN→TH for the bilingual `docs/` deliverables. Companion to `CONTEXT.md`
(English-only architectural glossary).

## Sources

- **PISA TH**: <https://pisathailand.ipst.ac.th/about-pisa/science_competency_framework/>
  (สสวท. / IPST). Authoritative for headline competencies *and* sub-ability verbs.
- **Waldorf TH**: anthrothailand.wordpress.com; ปัญโญทัย / Panyotai Waldorf School;
  thecitizen.plus reporting; Inodai Waldorf School (Wikipedia).
- **Construct (no canonical source)**: marked _construct_ — coined for this project.

## Translation policy

- Write Thai from comprehension of the English passage, not by term substitution.
  Glossary is a fallback reference, not a template.
- Headline competencies S1–S3, E1–E3: **PISA TH verbatim** (canonical).
- Sub-ability prose: prefer PISA TH verb vocabulary where available.
- Register: mid-formal academic. No informal particles (ก็เลย, น้า, นะ),
  no contractions like "บอกได้เลย". `การ-` nominalisation OK where natural.
- Codes (S1–S3, E1–E3), proper nouns (PISA, OECD, Waldorf, Steiner): stay English.
- Framework jargon: bilingual first mention `ระดับมาโคร (ภาพรวม)`, inline Thai
  thereafter.
- Chips translated; CSS class preserved.

## PISA competencies (verbatim from PISA TH)

| Code | English (framework.html) | Thai (PISA TH verbatim) |
|------|--------------------------|--------------------------|
| S1 | Explain phenomena scientifically | การอธิบายปรากฏการณ์ในเชิงวิทยาศาสตร์ |
| S2 | Construct/evaluate designs for scientific enquiry; interpret data critically | การออกแบบและประเมินกระบวนการสืบเสาะหาความรู้ทางวิทยาศาสตร์ และการแปลความหมายข้อมูลและใช้ประจักษ์พยานในเชิงวิทยาศาสตร์อย่างมีวิจารณญาณ |
| S3 | Research, evaluate and use scientific information for decisions and action | การศึกษาค้นคว้า ประเมิน และใช้ข้อมูลทางวิทยาศาสตร์เพื่อการตัดสินใจและการลงมือกระทำ |
| E1 | Explain the impact of human interactions with Earth's systems | การอธิบายผลกระทบจากการกระทำของมนุษย์ที่มีต่อระบบโลก |
| E2 | Make informed decisions to regenerate and sustain the environment | การตัดสินใจเพื่อการลงมือกระทำด้วยข้อมูล โดยใช้การประเมินแหล่งข้อมูลของประจักษ์พยานที่หลากหลาย |
| E3 | Demonstrate respect for diverse perspectives, and hope, in seeking solutions to socio-ecological crises | การแสดงถึงความมุ่งหวังและเคารพต่อมุมมองที่หลากหลายในการแสวงหาทางออก |

|---------|------|

| Procedural knowledge | ความรู้ด้านกระบวนการ |

| Epistemic (adjective, formal) | เชิงญาณวิทยา |
| Assessment framework | กรอบการประเมิน |
| Sustainability | ความยั่งยืน |
| Agency | การเป็นผู้นำในการเปลี่ยนแปลง |
| Science identity | อัตลักษณ์ทางวิทยาศาสตร์ |

**Epistemic context rule:** prefer `เชิงญาณวิทยา` for adjectival/stance uses
(`epistemic stance`, `epistemic responsibility`); prefer `เกี่ยวกับการได้มาของ
ความรู้` when the gloss aids comprehension or when paired with knowledge-type
labels (e.g. content / procedural / epistemic).
